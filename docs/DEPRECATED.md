## üéØ Why the Change?

### Reasons for Refocusing

1. **Accessibility** - Browser-based games reach more players without installation
2. **Cross-platform** - Works on any device with a modern browser
3. **Development Speed** - Faster iteration and deployment with web technologies
4. **Maintainability** - Single codebase is easier to maintain
5. **Immediate Playability** - Players can start instantly via URL

### What This Means

- **No installation required** - Just open the URL and play
- **Automatic updates** - Players always have the latest version
- **Mobile support** - Touch controls work on phones and tablets
- **Simpler contribution** - Easier for developers to get started
- **Faster development** - Focus on one technology stack
